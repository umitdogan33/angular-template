# AngularTemplate
Bu bir angular template uygulamasıdır

## Hazırlık
<ul>
  <li>console'u açıp <b>npm install</b> yazınız. Bu gerekli paketlerin kurulması için gereklidir.</li>
  <li>daha sonra <b>ng serve --o</b> yazarak projeyi çalıştırınız.</li>
</ul>
